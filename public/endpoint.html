API Endpoints

Root Table
Get /
Users Table
Get all users /users
Get by id /users/:id
register a user /register
post/login /login
Products Table
Get /products
Get by id /products/:id
Cart Table
Get /users/:id/cart
post/add to cart /users/:id/cart (the product id should be in the body aswell when posting in the backend)
delete on item from cart /users/:id/cart/:cartid
delete all items from cart /users/:id/cart
